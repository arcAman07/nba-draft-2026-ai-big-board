# Report Artifacts

Primary deliverable:

- `final_report.pdf`: final 59-page LaTeX research paper.

Source:

- `report.tex`: LaTeX source generated by `src/generate_latex_report.py`.

Companion artifacts:

- `final_report_latex.pdf`: duplicate of the final LaTeX PDF kept as an explicit LaTeX build artifact.
- `report.md`, `report.html`, `report.css`: legacy markdown/HTML report artifacts from the earlier generator. They are kept for traceability, but the LaTeX PDF is the final report.

Rebuild:

```bash
make report
```
