# Dossiers

This folder contains one markdown dossier per top-40 prospect.

Each dossier includes:

- Board rank and tier.
- Position, team/school, age, measurements, and key stats.
- Model median and uncertainty band.
- Consensus rank and spread.
- Historical comp floor, median, and ceiling.
- Best broad team fits.
- Sourced scouting synthesis.
- Frame-based film note where available.
- Final scouting verdict.

The dossiers are generated by `src/generate_report.py` and summarized more fully in the LaTeX paper at `report/final_report.pdf`.
