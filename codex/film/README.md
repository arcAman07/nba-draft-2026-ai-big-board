# Film

This folder stores the public-video and still-frame scouting archive.

## Structure

- `clips/`: short public clips imported from the local yt-dlp/ffmpeg archive.
- `frames/`: extracted still frames organized by prospect.
- `frames/_report_picks/`: selected frames embedded in the final LaTeX report.
- `notes/`: prospect film notes.
- `film_manifest.csv`: clip/frame manifest.

## Interpretation Rule

Frame-based observations are intentionally limited. They can support static claims about release point, body position, handle posture, gather height, finishing angle, or defensive stance in a sampled moment. They do not support confident claims about burst, processing speed, defensive timing, or live advantage creation.
